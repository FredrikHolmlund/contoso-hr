# Demoföretag – Medarbetarportal

En fiktiv medarbetarportal byggd som demo-miljö för att utveckla och visa upp AI-agenter — specifikt **HR-agenter**.

## Syfte

Sidan simulerar ett riktigt företags interna HR-portal och används som underlag när man bygger, testar och demonstrerar AI-agenter som hanterar vanliga HR-frågor, t.ex:

- Semesterregler och ledighetsansökan
- Förmåner och friskvårdsbidrag
- Sjukfrånvaro och karens
- Lön och utlägg
- IT-support och utrustning

Portalen ger agenten ett realistiskt sammanhang att arbeta i utan att röra riktiga system eller känslig data.

## Demo

**Live:** [fredrikholmlund.github.io/contoso-hr](https://fredrikholmlund.github.io/contoso-hr/)

## Teknik

- Ren HTML/CSS/JS — inga beroenden, ingen byggprocess
- Responsiv layout med sidebar-navigation
- FAQ-sektion med accordion-interaktion

## Struktur

```
index.html     # Portalen
sitemap.xml    # För sökmotorindexering
robots.txt     # Crawler-direktiv
```
